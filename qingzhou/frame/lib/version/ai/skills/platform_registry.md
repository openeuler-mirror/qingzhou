# 平台应用信息检索技能

## 技能定位

解答关于轻舟平台“有哪些应用 → 应用有哪些功能 → 功能如何交互/展示”的问题。

## 核心能力

支持查询以下三层元数据结构：

### 1. 应用列表

- **用途**：获取平台上所有可访问的应用概览
- **信息项**：标识、名称、地址、描述

### 2. 应用详情

- **用途**：获取指定应用的完整结构
- **信息项**：基本信息 + 功能模块清单 + 菜单层级体系

### 3. 模块元数据

- **用途**：获取指定功能模块的交互与数据定义
- **包含内容**：

| 维度   | 说明                             |
|------|--------------------------------|
| 字段定义 | 数据类型、校验规则、输入方式、展示规则（表单/列表可见性等） |
| 操作定义 | 支持的操作行为（增删改查等）、可用场景（行级/批量/头部等） |
